## Accessibility Features

### Screen Reader Support
- All input cells have clear labels in Column A
- Results have descriptive labels
- No color-only indicators (use icons too)

### High Contrast Mode
- Ensure text readable on all backgrounds
- Minimum contrast ratio: 4.5:1
- Test with Windows High Contrast

### Keyboard Navigation
- Ensure Tab order is logical (top to bottom)
- Input cells accessible via Tab key
- Dropdowns operable with keyboard
